# ERP_report_generator
ERP data automatic report generation tool based on Python. Realize data cleaning, analysis and Excel report one-click output with GUI interface.
This is an automated ERP report generation tool developed in Python, which supports automatic reading, cleaning, and basic statistical analysis of raw ERP data, a visual GUI interface, as well as one-click export of standardized Excel reports. 
Current limitations:
1. The program only supports Excel files exported from a specific ERP system with a fixed format; other formats or custom headers are not compatible.
2. Only basic statistical calculations are implemented; advanced indicators such as month-on-month or year-on-year growth rates are not yet supported.
3. Generated reports focus on practicality and data integrity, with limited formatting and no advanced styling or chart beautification.
The project aims to simplify repetitive daily report work and improve data processing efficiency, and can be further extended based on actual business requirements.
